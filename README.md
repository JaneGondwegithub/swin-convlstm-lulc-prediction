## Data Availability

Sample data is provided in `data/sample/` for demonstration purposes.

The full dataset (including LULC maps and spatial drivers) is not included in this repository due to size constraints.

To run the model:
1. Download or prepare the full dataset
2. Place the data in:
   data/processed/
3. Update file paths in the Config section if necessary

Full dataset is available upon request or via external repository (e.g., Google Drive or Zenodo).
